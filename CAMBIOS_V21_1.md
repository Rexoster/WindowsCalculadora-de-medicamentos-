# Cambios v21.1 · Rama correcta de este chat

Esta versión fue reconstruida directamente sobre la v20.2
`Gestograma/FUM responsivo` de esta conversación.

Incluye únicamente:

- Encabezados de la tabla fijos al desplazarse verticalmente.
- Desplazamiento horizontal sincronizado entre encabezado y filas.
- “Ver más” en columnas cuyo contenido realmente se desborda.
- Ventana desplazable para consultar el texto completo.

No incluye:

- Catálogos creados a partir del PDF de enfermedades infecciosas.
- Medicamentos agregados en otro chat.
- Archivos de datos procedentes de otra rama.

Los únicos ejemplos precargados siguen siendo:

1. Paracetamol adulto.
2. Paracetamol pediátrico.
3. Insulina basal.

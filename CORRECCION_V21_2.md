# Corrección v21.2 · Unidad visible en la dosis

- La celda Dosis ahora muestra también `doseUnit`.
- Una dosis numérica se presenta como `5 mg`.
- Si el texto ya incluye la unidad, no se duplica.
- Las indicaciones textuales muestran la unidad como metadato:
  `1 tableta · Unidad: mg`.
- Las dosis especiales y pediátricas mantienen el formato:
  `10 mg/kg`.
- Las dosis interactivas mantienen su rueda y el formato:
  `0.3 UI/kg`.
- “Ver más” sigue apareciendo únicamente cuando el contenido
  realmente sobrepasa el espacio disponible.
